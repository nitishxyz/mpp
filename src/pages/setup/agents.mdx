# Use with an agent [Point your AI agent at paid services—no setup required]

Using Claude, Codex, or another AI agent? You can give it a funded wallet so it pays for MPP-enabled services itself. [Presto](https://github.com/tempoxyz/presto) is the CLI that does that. Run the steps below; after that your agent can call paid APIs on its own.

<AgentTabs />

The **Connect wallet** step (`presto login`) is the only one that needs you—you complete it once in your browser. After that, the agent uses Presto for payments. For a full reference you can give your agent, see [setup.md](/setup.md).
