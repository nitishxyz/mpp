# `Method.tempo.charge` [One-time payments]

Creates a Tempo charge payment method for client-side transaction signing.

## Usage

```ts twoslash
// @noErrors
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

const fetch = Fetch.from({
  methods: [
    // [!code focus:start]
    tempo.charge({ account }),
    // [!code focus:end]
  ],
})

const response = await fetch('https://mpp.dev/api/ping/paid')
```

### With polyfill

```ts twoslash
// @noErrors
import { Fetch, tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const account = privateKeyToAccount('0x...')

Fetch.polyfill({
  methods: [tempo.charge({ account })],
})
```

## Return Type

```ts
import type { Method } from 'mpay'

type ReturnType = Method.Client
```

## Parameters

### account (optional)

- **Type:** `Account`

Account to sign transactions with. Can be overridden per-call via context.

```ts twoslash
// @noErrors
import { tempo } from 'mpay/client'
import { privateKeyToAccount } from 'viem/accounts'

const method = tempo.charge({
  account: privateKeyToAccount('0x...'), // [!code focus]
})
```

### client (optional)

- **Type:** `(chainId: number) => Client`

Function that returns a viem client for the given chain ID. Mutually exclusive with `rpcUrl`.

### rpcUrl (optional)

- **Type:** `{ [chainId: number]: string }`

RPC URLs keyed by chain ID. Mutually exclusive with `client`.

```ts twoslash
// @noErrors
import { tempo } from 'mpay/client'

const method = tempo.charge({
  // [!code focus:start]
  rpcUrl: {
    123456: 'https://rpc.tempo.xyz',
  },
  // [!code focus:end]
})
```
