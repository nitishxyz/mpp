# `tempo`

Creates a Tempo payment method for server-side payment handling.

## Usage

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  // [!code focus:start]
  method: tempo(),
  // [!code focus:end]
})
```

### With Fee Payer

Provide a fee payer account to cover transaction fees on behalf of users.

```ts twoslash
import { privateKeyToAccount } from 'viem/accounts'
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  method: tempo({ 
    feePayer: privateKeyToAccount('0x...'), // [!code focus]
  }), 
})
```

## Return Type

```ts
import type { Method } from 'mpay'

type ReturnType = Method.Server
```

## Parameters

### feePayer (optional)

- **Type:** `Account`

Fee payer account for covering transaction fees.

```ts twoslash
import { privateKeyToAccount } from 'viem/accounts'
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  method: tempo({ feePayer: privateKeyToAccount('0x...') }), // [!code focus]
})
```

### rpcUrl (optional)

- **Type:** `{ [chainId: number]: string }`

RPC URLs keyed by chain ID.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  method: tempo({
    // [!code focus:start]
    rpcUrl: {
      1337: 'http://localhost:8545',
      4217: 'https://rpc.tempo.xyz',
    },
    // [!code focus:end]
  }),
})
```

### testnet (optional)

- **Type:** `boolean`

Testnet mode.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const mpay = Mpay.create({
  method: tempo({
    testnet: true, // [!code focus]
  }),
})
```
