# `Mpay.create`

Creates a server-side payment handler from a method.

## Usage

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const payment = Mpay.create({
  methods: [tempo.charge()],
})
```

### With Custom Transport

Use a custom transport for non-HTTP environments like MCP servers.

```ts twoslash
import { Mpay, tempo, Transport } from 'mpay/server'

const payment = Mpay.create({
  methods: [tempo.charge()],
  transport: Transport.mcpSdk(),
})
```

## Return Type

```ts
import type { Method } from 'mpay'
import type { Mpay, Transport } from 'mpay/server'

type ReturnType = Mpay<[Method.Server], Transport.Http>
```

The returned object includes the method's intent functions (e.g., `charge`) which can be called to handle payment requests.

## Parameters

### methods

- **Type:** `readonly Method.Server[]`

Array of payment methods (for example, `[tempo.charge()]`).

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const payment = Mpay.create({
  // [!code focus:start]
  methods: [tempo.charge()],
  // [!code focus:end]
})
```

### realm

- **Type:** `string`

Server realm (e.g., hostname). Used to identify the server in payment challenges.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const payment = Mpay.create({
  methods: [tempo.charge()],
  realm: 'mpp.dev', // [!code focus]
})
```

### secretKey

- **Type:** `string`

Secret key for HMAC-bound challenge IDs. Enables stateless verification—allows the server to verify that a challenge was issued by itself without storing state. Defaults to `'tmp'`.

```ts twoslash
import { Mpay, tempo } from 'mpay/server'

const payment = Mpay.create({
  methods: [tempo.charge()],
  secretKey: process.env.MPAY_SECRET_KEY!, // [!code focus]
})
```

### transport (optional)

- **Type:** `Transport`
- **Default:** `Transport.http()`

Transport to use for handling payment requests. Defaults to HTTP transport.

```ts twoslash
import { Mpay, tempo, Transport } from 'mpay/server'

const payment = Mpay.create({
  methods: [tempo.charge()],
  transport: Transport.mcp(), // [!code focus]
})
```
